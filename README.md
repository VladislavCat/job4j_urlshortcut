# Сервис на REST API с созданием укорачивателя ссылок
## Стек технологий 
+ Java 17.0.2
+ Maven 3.8.5
+ Spring Boot 2.5.2
+ Postgres SQL 14.3
+ Hibernate 5.6.11 Final
+ Liquibase Maven Plugin 3.6.2
+ Spring Security 
+ REST API
 
## Для запуска
Для запуска приложения потребуется:
+ Java 17 и выше
+ Maven 3.8,
+ PostgresSQL 14

В первую очередь, создаем базу данных следующей командой:
```create database url```

Создаем таблицы с помощью Liquibase, далее производим запуска самого приложения командой

```mvn spring-boot:run```

## Использование 
Сперва вам нужно зарегистрировать приложение по запросу POST **/registration, передав в теле POST HTTP запроса имя вашего сайта.

Приложение вернет в теле ответа ваш логин и пароль для дальнейшего использования. 

С помощью этих данных вы должны авторизоваться по запросу POST на адрес **/login, передав в теле запроса логин и пароль. Стоит заметить, что в заголовке ответа авторизации будет указан ваш токен, который вы обязаны передавать на сервис в заголовке ваших запросов.  

После этого вы можете отправить запрос POST по ссылке **/convert. В теле запроса должна быть указана полная ссылка на ту или иную страницу. В теле ответа сервис пришлет вам ключ, который в дальнейшем вы можете использовать для redirect'a.

С помощью запроса GET по ссылке **/redirect/{key} вы можете получить в теле ответа оригинальную ссылку на страницу. Данный запрос не требует авторизации.

С помощью запроса GET по ссылке **/statistic вы можете получить статистику, а именно, сколько раз обращались к каждому ключу.
